OWL2DIG

    A Java library and tool for converting OWL to DIG

Apex Data and Knowledge Management Lab

Department of Computer Science and Engineering

Shanghai Jiaotong University, Shanghai

Overview

OWL2DIG is a Java library and tool for converting OWL to DIG (Description Logic Interface in a XML format).

Installation

  • unzip the distribution file into a convenient directory, OWL2DIGROOT.
  • the executable batch file owl2dig.bat is stored in OWL2DIGROOT/bin.
