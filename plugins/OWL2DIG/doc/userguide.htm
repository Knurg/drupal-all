Name:

    owl2dig - Translator from OWL to DIG

SYNOPSIS:

    owl2dig owlfile digfile [kb-uri]

Description:

      owl2dig reads the input owlfile into a Jena OntModel and converts it into DIG by translating every RDF triple of the OWL model. The input OWL file must conform to OWL-DL standard which means that an instance and a class can never be related by a property except for the RDF:type property.

      After executing the command, the output file can be sent to a DIG-compliant reasoner.

Parameters:

    owlfile is the input OWL file name.

    digfile is the output DIG file name.     

    kb-uri is the knowledgebase identifier which is defined in DIG 1.1. If not specified, the tells element of the DIG document won't have the uri attribute.
